# IndieLaunch

Help indie SaaS founders get their first 100 paying users.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Features

- Hero section with email waitlist
- Smart pricing calculator
- Social proof testimonials
- Dark theme with IBM Plex Mono
- Responsive design

## Pages

- `/` - Home with waitlist and features
- `/pricing` - Interactive pricing calculator

## Tech Stack

- Vite + React 18
- React Router v6
- Inline styles for speed

Ready to ship! 🚀
